# MeetNow

MeetNow提供对接亿联会议的简易API，主要包括以下功能。

- 账户登录

- 媒体入会

- 会议控制

  - 人员控制
  - 录制控制
  - 直播控制

## 状态: Beta

当前代码状态提供上述功能，虽然API已经基本稳定，但请注意仍然存在API小幅修改的可能。

## 已知问题

- 目前不支持亿联账号登录，对接亿联会议采用匿名登录方式。

- 目前只支持单匿名用户/单会议

## 使用建议

- 媒体入会使用WebRTC API，故不支持 Internet Explorer 和 小程序 进行媒体入会

- 小程序入会请使用指定小程序组件

- 引入 webrtc-adapter 以保证代码更好的兼容性
